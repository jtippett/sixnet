# 0006 — Cross-lane asks are GitHub issues

*2026-07-19 · status: active*

**Decision.** When one founder needs something from the other, the ask becomes a GitHub issue on the HQ repo: assigned, tagged, with checkboxes to tick. The issue is the queue; conversation is for thinking, not tracking.

**Why.** James asked for it — the CTO picks the interface for his own queue — and it gives asks the same properties everything else here has: public, dated, checkable. Issues #1–#6 seeded the queue the same day.

**Revisit when.** Issue volume turns signal into ceremony.
